## Main rule

Do not treat this as a generic seafood-site prompt pack.

This project already has hard business and service truth:

- address-first delivery
- pickup as a separate path
- delivery-to-time with `±15 minutes`
- current manual bridge before `iiko`
- future multi-point / franchise-ready routing
- crayfish and crab as premium menu core, without turning homepage into category taxonomy

The build should upgrade the existing site into a stronger real site pass without losing that truth.
