# Visual References

## Как использовать этот файл

Это не список для копирования.

Это набор `уровней силы`, на которые нужно ориентироваться по:

- смелости;
- art direction;
- композиции;
- целостности;
- вкусу.

## Primary references

### 1. High-end editorial hospitality / food brand sites

Ищем от них:

- authored homepage;
- сильную типографику;
- вкусный image-first narrative;
- не-шаблонную композицию;
- высокий taste level.

Что брать:

- ощущение бренда как культурного объекта, а не только commerce surface;
- ритм секций;
- выразительный use of negative space;
- дорогую визуальную тишину.

### 2. Premium commerce with editorial control

Ищем от них:

- как продавать товар не как marketplace;
- как соединять appetite и clarity;
- как держать product route премиальным без потери точности.

Что брать:

- hierarchy;
- surface discipline;
- точный use of data blocks;
- product desire before controls.

### 3. Strong map/service interfaces inside premium brands

Ищем от них:

- как показывать service UI без ощущения админки;
- как встроить карту в дорогой consumer-facing experience.

Что брать:

- curated map surface;
- restrained overlays;
- спокойные state labels;
- дорогую точность.

## Anti-references

### 1. Generic luxury dark restaurant templates

Нельзя брать:

- predictable centered hero;
- huge headline + smoky image + same cards below;
- flat “дорогой” dark mode without hierarchy;
- одинаковые rounded blocks везде.

### 2. Marketplace / aggregator feel

Нельзя брать:

- dense catalog grids;
- small product cards as the main language;
- chip overload;
- delivery-app heuristics.

### 3. SaaS / admin UI under a dark skin

Нельзя брать:

- dashboard layouts;
- panel stacks;
- utilitarian tables as visual base;
- form-first thinking on premium routes.

## If uncertain, choose this direction

Если есть сомнение между:

- more functional but ordinary;
- или более authored and premium,

выбирай authored and premium, если route остается usable.
